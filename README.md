# LilBuddy
A Desktop Pet that acts as your buddy on your screen with helpful features! 
You can customize it's animations to whatever you want and you can add in your own scripts modularly to improve what it can do:D

Out-of-the-box Features:
1. Pomodoro Timer
2. Transcribe (offline mic/WAV via Vosk)
3. Feed and Talk to pet
4. Notes
